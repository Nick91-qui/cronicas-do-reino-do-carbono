# Mapa de Telas

## Objetivo do documento

Este documento descreve como o visual deve se distribuir nas telas principais do MVP.

## Telas centrais

### 1. Entrada no jogo

Deve comunicar:

- identidade do capítulo ativo;
- progresso geral;
- acesso rápido à próxima fase;
- sensação de campanha em andamento.

### 2. Tela de capítulo

Deve comunicar:

- arco narrativo do capítulo;
- fases desbloqueadas;
- progressão linear;
- recompensas já conquistadas.

### 3. Tela de fase

É a tela visual mais importante do MVP.

Ela deve incluir:

- cabeçalho de fase;
- lore panel;
- missão;
- laboratório de síntese;
- action bar;
- cartas disponíveis;
- justificativa por propriedades;
- feedback final.

### 4. Coleção de cartas

Deve funcionar como:

- memória visual do progresso;
- consulta pedagógica;
- comparação entre moléculas desbloqueadas.

### 5. Perfil e progresso

Deve resumir:

- títulos desbloqueados;
- cartas obtidas;
- estado geral do capítulo.

## Ordem de prioridade visual

Para o MVP, a prioridade de refinamento visual deve ser:

1. tela de fase;
2. carta de molécula;
3. tela principal do jogo;
4. tela de capítulo;
5. coleção;
6. perfil.

## Regra de consistência

As telas devem compartilhar:

- mesma família de painéis;
- mesma semântica de cores;
- mesma linguagem de botões e destaque;
- mesma lógica de feedback visual.
